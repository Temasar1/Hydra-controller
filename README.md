# Hydra Controller

Multi-node management interface for Cardano Hydra Layer 2 operations.

## Setup

```sh
git clone <YOUR_GIT_URL>
cd Hydra-controller
npm i
npm run dev
```

## Tech

- Next.js (App Router)
- TypeScript
- React
- Tailwind CSS

## Deploy

Build and run:

```sh
npm run build
npm start
```

Or deploy to any Node-compatible host (Vercel, Railway, etc.) using the same build command.
